# Figma Assets

Place the exported `.fig` file for the SEEDRA Ventures Brand Identity System here.

## File naming convention

- `seedra-brand-identity-system.fig` — Main brand file with logo, color styles, text styles, and component library.

## How to export

1. Open the Figma file.
2. Go to **File > Save local copy...**
3. Rename to `seedra-brand-identity-system.fig` and drop it into this folder.
